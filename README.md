# Installation

Available as an [npm package](https://www.npmjs.com/package/bailu-ui)

```bash
// with npm
npm install bailu-ui

// with yarn
yarn add bailu-ui

// with pnpm
pnpm add bailu-ui
```

# Examples

```typescript
import { createApp } from 'vue'
import BaiLuUI from 'bailu-ui';
import App from './App.vue';
import 'bailu-ui/css';

const app = createApp(App);
app.use(BaiLuUI);
app.mount('#app');
```
